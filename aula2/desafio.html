HTML


HTML (HyperText Markup Language) é a Linguagem de Marcação padrão para criar a estrutura de páginas web.

O Html define elementos, parágrafos, títulos, imagens e links, organizando o conteúdo para navegadores.
Não é uma linguagem de programação, mas sim a base estrutural, usada com CSS (visual) e JavaScript (interatividade)
